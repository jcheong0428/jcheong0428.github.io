How to use
×

The Video Event Segmentation Tool (VEST) is an open-source software for annotating temporal events in video clips.

You can annotate event "segments" with start & end times or event "points" that indicate only the start time of an event.

Basic instructions

  1. Load your video file using the [Choose file] button

  2. Define the name of the event segments and points using the [Add segment buttons] and [Add point buttons]. This will create buttons with your event names on them.

  3. Click your label buttons to annotate segments or events as you watch your video.

  4. When you are finished, save your annotations into a json file by using the [Download results] button.

Feedback

Please leave feedback in the Github repository.

Privacy & security

When using VEST, your videos and annotations are processed in your browser. This means your videos and annotations are just as private as being on your own laptop. This also means that you will lose any video and annotation progress when you close your browser. Make sure you frequently save your data!
